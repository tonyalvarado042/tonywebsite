## 8. Decisiones de diseño pendientes

Estas decisiones deben tomarse antes de iniciar el código:

1. **Paleta de color final:** el prototipo anterior usa cian (`#38bdf8`). El CLAUDE.md menciona "morado y blanco" como posibles colores de marca. Confirmar con el cliente cuál es el acento oficial.
2. **Tipografía:** definir si se usa Inter, DM Sans u otra fuente con carácter deportivo y premium.
3. **Foto del hero:** confirmar si `principal.PNG` es la foto oficial para el hero o si existe una versión de mayor resolución.
4. **Logotipo:** texto simple "TonyAlvarado" o logotipo gráfico. [INFORMACIÓN PENDIENTE]
