## 30. Forma correcta de trabajar este proyecto

No ejecutar el DAB completo de una sola vez.

Orden correcto:

1. Auditar archivos actuales.
2. Validar assets disponibles.
3. Validar información faltante.
4. Definir arquitectura web.
5. Crear o mejorar prototipo.
6. Migrar a Next.js solo cuando el usuario lo autorice.
7. Implementar SEO técnico.
8. Implementar formularios.
9. Crear PR package.
10. Preparar autoridad externa.
11. Preparar optimización para IA.
12. Preparar mantenimiento.

- Qué se hizo.
- Qué archivos se tocaron.
- Qué falta.
- Qué requiere validación.
